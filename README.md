# Portfolio

O projeto é o código fonte do meu portifólio (https://johnisonf.com). Esse código foi 100% utilizando a minha experiência própria com frontend sem qualquer ajuda de IA.

# Arquivos importantes e Pastas

`style.css`: Arquivo onde tenho estilos base e importo todos os outros arquivos css dentro da passa `assets/css`.

`/assets`: Pasta contendo todos os arquivos de css, funções de javascript e imagens (não listadas).

`functions.js`: Arquivo principal javascript para rodar as funções exportadas de arquivos dentro da pasta `assets/js`.
